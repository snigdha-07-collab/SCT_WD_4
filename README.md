# ✅ TaskFlow - Smart To-Do Web Application

A modern and responsive To-Do Web Application built using **HTML, CSS, and JavaScript**. TaskFlow helps users efficiently manage daily tasks with a clean interface, smart organization, and persistent storage.

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed
- 📅 Set due date
- ⏰ Set due time
- 🔴🟡🟢 Priority Levels (High, Medium, Low)
- 🔍 Live Search
- 📂 Filter Tasks (All, Pending, Completed)
- 📊 Progress Tracker
- 📈 Task Statistics
- 🌙 Dark / Light Mode
- 🕒 Live Date & Clock
- 💬 Random Motivational Quotes
- 💾 Local Storage Support
- 📱 Fully Responsive Design
- ✨ Smooth Animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Font Awesome

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your browser.
4. Start managing your daily tasks!
